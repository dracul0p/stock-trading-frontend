# 📈 Stock Trading Frontend (Zerodha UI Clone)

This project is a **frontend clone of the Zerodha website** built to practice **React.js, Tailwind CSS, and modern frontend development**.

It focuses on building reusable components, responsive layouts, and a clean UI similar to a real trading platform website.

🌐 **Live Demo**
👉 [https://zerodha-frontend-practice.netlify.app/](https://zerodha-frontend-practice.netlify.app/)

---

# 🚀 Features
* Modern **React.js component-based architecture**
* **Responsive UI** for desktop and mobile
* Styled with **Tailwind CSS**
* Multiple landing pages similar to Zerodha
* Clean and modular folder structure
* Deployed using **Netlify**

---

# 🛠️ Tech Stack
* **React.js**
* **Tailwind CSS**
* **JavaScript (ES6)**
* **HTML5**
* **Netlify (Deployment)**

---

# ⚡ Installation

Clone the repository
```bash
git clone https://github.com/yourusername/stock-trading-frontend.git
```

Go to project directory
```bash
cd stock-trading-frontend
```

Install dependencies
```bash
npm install
```

Run the development server
```bash
npm start
```
